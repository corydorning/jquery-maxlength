jQuery MaxLength Plugin

MaxLength is a jQuery plugin that only allows a set number of characters to be entered into a <textarea>. It has built-in feature detection and is not used if it is supported natively by the browser.

MaxLength Plugin Examples

This textarea will only accept 10 characters

MaxLength Plugin Dependencies

To use maxlength, it is recommended that you use jQuery version 1.8+. Reference the jQuery library followed the maxlength JavaScript file:

<!-- load jQuery library -->
<script src="/common/js/libs/jquery-1.8.min.js"></script>

<!-- load restricted plugin -->
<script src="jquery-maxlength.js"></script>

MaxLength Plugin Usage

Simply reference the plugin via an appropriate <textarea> selector:

$('textarea').maxlength();
